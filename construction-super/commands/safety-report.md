# Command: /construction:safety-report

Generate a weekly jobsite safety summary report.

## Output Format
```
WEEKLY SAFETY REPORT
Project: [name] | Week of: [date range]
Safety Manager/Superintendent: [name]

INCIDENT SUMMARY:
☐ Zero incidents this week
☐ [X] incidents (see details below)

SAFETY STATISTICS:
Total Man-Hours This Week: [X]
Total Man-Hours to Date: [X]
Recordable Incidents to Date: [X]
TRIR (Total Recordable Incident Rate): [X]
Lost Time Incidents to Date: [X]
Days Since Last Recordable: [X]

TOOLBOX TALKS CONDUCTED:
Date       Topic                          Attendance
──────────────────────────────────────────────────────
[date]     [topic]                        [X] workers
[etc.]

SITE INSPECTIONS:
Date       Inspector       Findings              Corrected?
──────────────────────────────────────────────────────────────
[date]     [name]          [finding]              [Y/N]

HAZARDS IDENTIFIED & CORRECTED:
• [Description of hazard and corrective action taken]

TRAINING COMPLETED:
• [Training topic, date, number of attendees]

NEAR-MISS REPORTS:
• [Description of near-miss events]

FOCUS AREA FOR NEXT WEEK: [topic]
```
