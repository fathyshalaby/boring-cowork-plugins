# Command: /construction:rfi

Generate a formal Request for Information (RFI) to the architect/engineer.

## Output Format
```
REQUEST FOR INFORMATION (RFI)
RFI #: [number] | Date: [date]
Project: [name] | Contract #: [number]
From: [Contractor, name] | To: [Architect/Engineer, name]

DRAWING REFERENCE: Sheet [number], Detail [number]
SPEC REFERENCE: Section [number], Paragraph [number]

QUESTION:
[Clear, specific question about the design intent or discrepancy]

SUGGESTED RESOLUTION:
[Contractor's proposed solution, if applicable]

COST/SCHEDULE IMPACT:
☐ No impact anticipated
☐ Potential cost impact: $[estimate]
☐ Potential schedule impact: [X] days

RESPONSE NEEDED BY: [date]
(Note: Delayed response may impact the critical path.)

ATTACHMENTS:
• [Annotated drawing, photo, sketch]

─── FOR DESIGN TEAM USE ───
Response: _______________________________________________
Responded By: ______________ Date: ______________
```
