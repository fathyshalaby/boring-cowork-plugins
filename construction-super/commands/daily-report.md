# Command: /construction:daily-report

Generate a professional superintendent's daily report.

## Output Format
```
DAILY CONSTRUCTION REPORT
Project: [name] | Date: [date] | Report #: [number]
Superintendent: [name]

WEATHER: High: [X]°F | Low: [X]°F | Conditions: [clear/rain/wind]
Weather Delay: [Yes/No] | Hours Lost: [X]

SUBCONTRACTORS ON SITE:
Trade                   Company              Headcount
────────────────────────────────────────────────────────
[trade]                 [company]            [X]
[etc.]
TOTAL MANPOWER: [X]

WORK PERFORMED TODAY:
[Detailed description by trade and area/floor]

MATERIAL DELIVERIES:
• [Material, quantity, supplier]

INSPECTIONS:
• [Type] – [Pass/Fail/Pending] – Inspector: [name]

EQUIPMENT ON SITE:
• [Equipment type, company, hours used]

SAFETY OBSERVATIONS:
• [Observations, toolbox talk topic, any incidents]

ISSUES / DELAYS / NOTES:
• [Description of any problems, RFIs submitted, change orders pending]

PHOTOS: [X] photos attached
```
