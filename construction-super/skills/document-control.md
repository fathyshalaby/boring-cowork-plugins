# Construction Document Control
Maintain organized, current project documentation:

### Critical Project Documents
| Document | Purpose | Update Frequency |
|----------|---------|-----------------|
| Contract drawings (plans & specs) | Design intent | As revised (track revisions!) |
| Submittals | Product/material approval | Per specification section |
| RFIs | Clarify design intent | As needed |
| Change Orders | Scope/cost/time changes | As negotiated |
| Daily Reports | Daily record of work, weather, manpower | Daily (end of each day) |
| Meeting Minutes | Record of decisions | Weekly (within 48 hours of meeting) |
| Schedule Updates | Current project timeline | Weekly or biweekly |
| Pay Applications | Monthly billing | Monthly |
| Inspection Reports | Code compliance verification | Per inspection event |

### Daily Report Essentials
Every superintendent daily report must include:
- Date, weather (temp high/low, conditions), and any weather delays.
- Subcontractors on site with headcount.
- Work performed by each trade (location and description).
- Materials delivered.
- Inspections conducted and results.
- Visitors to the site.
- Safety observations or incidents.
- Issues, delays, or potential problems.

### Change Order Management
- Never perform extra work without a signed change order or at minimum a written COP (Change Order Proposal).
- Document the directive: who asked for it, when, and why.
- Track costs meticulously: labor hours, material invoices, equipment rental, and subcontractor quotes.
- Include time impact: does this change affect the critical path? If so, request a time extension.
