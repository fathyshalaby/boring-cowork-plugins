# Conformity Assessment Modules
Determine the correct conformity assessment procedure:

### The "Modules" Concept
The EU defines various modules (A through H) for assessing conformity. The applicable Directive/Regulation dictates which modules you are legally allowed to use, based on product risk.

| Module | Name | Description | Notified Body? |
|--------|------|-------------|----------------|
| **A** | Internal Production Control | Manufacturer acts alone (self-certification). Compiles file, signs DoC. | NO |
| **B** | EU-Type Examination | NB examines technical design and issues an EU-Type Examination Certificate. | YES |
| **C** | Conformity to Type | Manufacturer ensures products match Module B type. | NO |
| **D** | Quality Assurance (Production) | NB audits manufacturer's production QMS (e.g., ISO 9001). | YES |
| **E** | Quality Assurance (Product) | NB audits final product inspection QMS. | YES |
| **F** | Product Verification | NB verifies and tests every single product or statistical batches. | YES |
| **G** | Unit Verification | NB tests the specific individual product (good for 1-off custom builds). | YES |
| **H** | Full Quality Assurance | NB audits full QMS (Design + Production). Covers Module B+D. | YES |

### When is a Notified Body Required?
- High-risk products (e.g., Annex IV machinery like circular saws, Class IIa/IIb/III medical devices). 
- When Harmonized Standards do not exist or were not fully applied.
- **Notified Body (NB)**: An independent organization designated by an EU country to assess conformity before being placed on the market. Listed on the EC's NANDO database.
- If an NB is involved in production control, their 4-digit ID number must appear under the CE mark (e.g., CE 0123).
