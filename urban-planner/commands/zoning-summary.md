# Command: /planning:zoning-summary

Generate a zoning district summary for a specific zone.

## Output Format
```
ZONING DISTRICT SUMMARY
District: [code] — [name]
Jurisdiction: [municipality]

PURPOSE: [intent statement]

PERMITTED USES: [list]
CONDITIONAL USES: [list]

DIMENSIONAL STANDARDS:
Min Lot Size: [X] sq ft | Min Lot Width: [X]'
Setbacks: Front: [X]' | Side: [X]' | Rear: [X]'
Max Height: [X]' / [X] stories
Max Lot Coverage: [X]% | Max FAR: [X]
Max Density: [X] du/acre

PARKING REQUIREMENTS:
[Use type: X spaces per 1,000 sf / unit]

OVERLAY DISTRICTS: [if applicable]
```
