# Command: /planning:public-comment-response

Generate a response to a public comment on a planning application.

## Output Format
```
PUBLIC COMMENT RESPONSE
Project: [name] | Case #: [number]
Commenter: [name] | Date Received: [date]

COMMENT SUMMARY:
[Brief summary of the concern raised]

STAFF RESPONSE:
[Factual response addressing the concern, referencing applicable
standards, policies, or conditions of approval]

ACTION TAKEN:
☐ No change to recommendation
☐ Additional condition of approval added
☐ Project modified to address concern

Response Provided By: [name, title]
Date: [date]
```
