# Command: /planning:staff-report

Generate a planning staff report for a public hearing item.

## Output Format
```
STAFF REPORT
Case#: [number] | Hearing Date: [date]
Applicant: [name] | Property: [address]
Request: [description of request]

BACKGROUND:
[Site description, surrounding uses, zoning history]

ANALYSIS:
[Evaluation against applicable criteria/standards]

COMPREHENSIVE PLAN CONSISTENCY:
[How the request aligns with the comprehensive plan]

PUBLIC COMMENTS:
[Summary of comments received]

STAFF RECOMMENDATION:
☐ Approval  ☐ Approval with conditions  ☐ Denial

CONDITIONS OF APPROVAL (if applicable):
1. [condition]

Prepared by: [name, title]
```
