# Command: /planning:density-calc

Generate a development density and yield analysis.

## Output Format
```
DEVELOPMENT DENSITY ANALYSIS
Site: [address/parcel] | Zoning: [code]
Site Area: [X] acres ([X] sq ft)

ZONING PARAMETERS:
Max Density: [X] du/acre | Max FAR: [X]
Max Height: [X]' | Max Coverage: [X]%
Setbacks: F:[X]' S:[X]' R:[X]'

BUILDABLE AREA:
Gross Site: [X] acres
- Setbacks: [X] acres
- Environmental (flood/wetland): [X] acres
- Infrastructure (roads/utilities): [X] acres
= Net Buildable: [X] acres

DEVELOPMENT YIELD:
Max Units (by density): [X]
Max Floor Area (by FAR): [X] sq ft
Parking Required: [X] spaces
Open Space Required: [X] sq ft

ESTIMATED YIELD: [X] units / [X] sq ft
```
