# GIS & Spatial Analysis for Planning
Use GIS as a core planning tool:

### Essential GIS Data Layers
| Layer | Source | Use |
|-------|--------|----- |
| Parcels | County assessor | Ownership, land use, valuation |
| Zoning | Planning department | Regulatory framework |
| Future land use | Comprehensive plan | Long-range planning |
| Floodplains | FEMA | Development constraints |
| Topography/contours | USGS/LiDAR | Slope analysis, grading |
| Utilities (water, sewer) | Public works | Capacity analysis |
| Transportation | DOT/MPO | Network analysis |
| Census | US Census Bureau | Demographics, socioeconomics |
| Aerial imagery | County/commercial | Visual reference, change detection |

### Common Spatial Analyses
- **Suitability analysis**: Overlay multiple criteria to identify the best locations for a specific use.
- **Buffer analysis**: Identify parcels within a specified distance (e.g., 500 feet from a proposed rezoning for notification purposes).
- **Density mapping**: Calculate and visualize dwelling units per acre, population density, or employment density.
- **Buildout analysis**: Estimate development capacity remaining under current zoning.
- **Accessibility analysis**: Map distance/travel time to services (parks, transit, grocery stores) to identify equity gaps.
