# Zoning Codes & Land Use Regulation
Navigate zoning classifications and land use regulations:

### Common Zoning Districts
| District | Code | Typical Uses |
|----------|------|-------------|
| Single-Family Residential | R-1, R-2 | Detached homes, low density |
| Multi-Family Residential | R-3, R-4, R-5 | Apartments, townhouses |
| Commercial (Neighborhood) | C-1, B-1 | Retail, office, small-scale |
| Commercial (General) | C-2, B-2 | Larger retail, restaurants, services |
| Industrial (Light) | I-1, M-1 | Light manufacturing, warehousing |
| Industrial (Heavy) | I-2, M-2 | Heavy manufacturing, processing |
| Mixed Use | MU, MX | Residential + commercial combination |
| Planned Unit Development | PUD | Flexible, master-planned projects |
| Agricultural | A, AG | Farming, rural uses |

### Dimensional Standards
- **Setbacks**: Minimum distance from property line to building (front, side, rear).
- **Lot coverage**: Maximum % of lot area covered by buildings.
- **Floor Area Ratio (FAR)**: Total building floor area ÷ Lot area. Controls building bulk.
- **Height limits**: Maximum building height (feet and/or stories).
- **Density**: Maximum dwelling units per acre.

### Zoning Relief
- **Variance**: Relief from a specific dimensional standard. Must show hardship unique to the property.
- **Special/Conditional use permit**: Allows a use permitted only with conditions (approved by planning commission/board).
- **Rezoning**: Legislative change to the zoning map. Requires public hearing, comprehensive plan consistency.
