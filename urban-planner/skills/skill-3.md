# Development Review Process
Administer the development review process effectively:

### Review Types
| Application | Decision Maker | Criteria |
|------------|---------------|----------|
| Site plan review | Staff or planning commission | Compliance with zoning, design standards |
| Subdivision plat | Planning commission + governing body | Subdivision regulations |
| Conditional use permit | Planning commission or board | Special use criteria |
| Variance | Board of Zoning Appeals | Hardship standards |
| Rezoning | Governing body | Comprehensive plan consistency |
| PUD | Governing body | Master plan, public benefit |

### Site Plan Review Checklist
- Zoning compliance (use, setbacks, coverage, height, parking count)
- Access and circulation (curb cuts, drive aisles, fire access)
- Parking and loading (count, dimensions, ADA spaces, bicycle parking)
- Landscaping and screening (buffers, street trees, parking lot landscaping)
- Stormwater management (detention/retention, erosion control)
- Utility connections (water, sewer, electric — capacity available?)
- Signage (size, height, illumination per sign code)
- Lighting (photometric plan, dark sky compliance)
- Architecture (if design standards apply)

### Interdepartmental Review
- Route applications to: engineering/public works, fire, utilities, transportation, parks, environmental. Set clear review timelines (14-21 days).
