# Command: /fleet:driver-memo

Generate a driver communication memo for policy updates or safety reminders.

## Output Format
```
FLEET MEMO
To: All Drivers | From: Fleet Manager
Date: [date] | Subject: [topic]

[Clear, concise message body]

ACTION REQUIRED:
☐ [Specific action drivers need to take]

EFFECTIVE DATE: [date]
QUESTIONS: Contact [name] at [phone/email]

ACKNOWLEDGMENT:
I have read and understand this memo.
Driver Name: _____________ Date: _______
Signature: _____________
```
