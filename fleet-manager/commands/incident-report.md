# Command: /fleet:incident-report

Generate a fleet vehicle incident/accident report.

## Output Format
```
VEHICLE INCIDENT REPORT
Date/Time: [X] | Location: [address]
Unit: [ID] | Driver: [name]
Police Report#: [X]

INCIDENT DESCRIPTION:
[Narrative of events]

DAMAGE ASSESSMENT:
Company Vehicle: [description, estimated cost]
Other Vehicle(s): [description]
Property Damage: [description]
Injuries: [details or "None reported"]

PREVENTABILITY:
☐ Preventable  ☐ Non-preventable  ☐ Under review

CORRECTIVE ACTION:
[Driver coaching, training, disciplinary action]

INSURANCE CLAIM: ☐ Filed (Claim#: [X])  ☐ Not filed
```
