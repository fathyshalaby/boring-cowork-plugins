# Command: /facility:budget-report

Generate a monthly operating budget variance report for building ownership.

## Output Format
```
MONTHLY OPERATING BUDGET REPORT
Property: [Building Name] | Period: [Month Year]

EXECUTIVE SUMMARY:
[1-2 paragraph summary of financial performance and key variances]

INCOME:
Category          Budget      Actual      Variance    %
─────────────────────────────────────────────────────────
Base Rent         $[X]        $[X]        $[X]        [X]%
CAM Recovery      $[X]        $[X]        $[X]        [X]%
Other Income      $[X]        $[X]        $[X]        [X]%

OPERATING EXPENSES:
Category          Budget      Actual      Variance    %
─────────────────────────────────────────────────────────
Utilities         $[X]        $[X]        $[X]        [X]%
Janitorial        $[X]        $[X]        $[X]        [X]%
R&M               $[X]        $[X]        $[X]        [X]%
[etc.]

NET OPERATING INCOME:
Budget: $[X] | Actual: $[X] | Variance: $[X] ([X]%)

VARIANCE EXPLANATIONS:
[Explain any category >10% over budget]

CAPITAL PROJECTS UPDATE:
[Status of any active capital projects]
```
