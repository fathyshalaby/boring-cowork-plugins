# Command: /facility:incident-report

Generate a formal incident report for documenting property damage, injuries, security events, or safety hazards.

## Output Format
```
INCIDENT REPORT
Report #: [number] | Date of Incident: [date/time]
Date Reported: [date] | Reported By: [name]

LOCATION: [Building, Floor, specific area]
TYPE: [Injury / Property Damage / Security / Environmental / Other]

DESCRIPTION OF INCIDENT:
[Detailed chronological account of what happened]

PERSONS INVOLVED:
• Name: [name] | Contact: [phone/email] | Role: [tenant/employee/visitor]

WITNESSES:
• Name: [name] | Contact: [phone/email]

IMMEDIATE ACTIONS TAKEN:
• [What was done to address the situation]

ROOT CAUSE (if known):
[Analysis of why the incident occurred]

PREVENTIVE MEASURES:
[Steps to prevent recurrence]

FOLLOW-UP REQUIRED:
☐ Insurance claim filed
☐ Police report filed (Report #: _____)
☐ Medical attention provided
☐ Maintenance work order created (WO#: _____)

Prepared by: [name] | Title: [title] | Date: [date]
```
