# Command: /grants:abstract-draft

Generate a project abstract/summary for a grant proposal.

## Output Format
```
PROJECT ABSTRACT
Title: [project title]
PI: [name] | Institution: [name]
Funding Request: $[X] | Duration: [X years]

[Paragraph 1: Problem/need statement — what gap exists and why it matters]

[Paragraph 2: Project approach — what you will do, key methods, innovation]

[Paragraph 3: Expected outcomes and broader significance]

Key Words: [5-8 relevant terms]
```
