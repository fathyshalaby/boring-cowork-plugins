# Command: /grants:compliance-check

Generate a pre-submission compliance checklist.

## Output Format
```
PRE-SUBMISSION COMPLIANCE CHECKLIST
Funder: [name] | FOA: [number] | Deadline: [date]

FORMATTING:
☐ Page limits met (narrative: [X] pages)
☐ Font/margins per guidelines ([font], [size], [margins])
☐ File format correct (PDF, named per convention)

CONTENT:
☐ All required sections included
☐ Review criteria explicitly addressed
☐ Budget within award limits
☐ Budget justification complete

REQUIRED DOCUMENTS:
☐ Biosketches/CVs for all key personnel
☐ Letters of support/collaboration
☐ Data management plan
☐ Human subjects / IRB documentation
☐ Institutional certifications

ADMINISTRATIVE:
☐ Grants.gov / portal registration current
☐ Authorized organizational representative (AOR) notified
☐ Internal review/routing completed
☐ Submission planned 48+ hours before deadline

STATUS: ☐ Ready to submit  ☐ Items outstanding
```
