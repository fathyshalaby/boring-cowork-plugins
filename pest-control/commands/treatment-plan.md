# Command: /pest:treatment-plan

Generate a treatment plan proposal for a client.

## Output Format
```
TREATMENT PLAN
Client: [name] | Address: [address]
Date: [date] | Inspector: [name]

PEST(S) IDENTIFIED: [species, population level, evidence found]

INSPECTION FINDINGS:
[Detailed description of pest activity, entry points, conducive conditions]

RECOMMENDED TREATMENT:
Phase 1: [Initial treatment — method, products, areas]
Phase 2: [Follow-up treatment — timing, method]
Phase 3: [Ongoing monitoring/prevention]

EXCLUSION RECOMMENDATIONS:
• [Entry points to seal, repairs needed]

CLIENT PREPARATION REQUIRED:
• [What the client needs to do before treatment]

ESTIMATED COST: $[X]
SERVICE WARRANTY: [terms]

Signature: _____________ Date: _________
```
