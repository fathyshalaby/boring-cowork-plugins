# Command: /pest:service-report

Generate a service visit report for a client.

## Output Format
```
SERVICE REPORT
Client: [name] | Account #: [number]
Address: [address]
Date: [date] | Tech: [name] | Cert #: [number]
Service Type: ☐ Regular ☐ Initial ☐ Callback ☐ Re-treat

CONDITIONS OBSERVED:
[Pest activity, sanitation issues, structural deficiencies]

SERVICES PERFORMED:
• [Treatment/action with specific locations]

PRODUCTS APPLIED:
Product          EPA Reg#     Rate        Location       Amount
────────────────────────────────────────────────────────────────
[product]        [reg]        [rate]      [where]        [qty]

MONITOR RESULTS:
[Trap/monitor catches since last visit]

RECOMMENDATIONS FOR CLIENT:
• [Sanitation, exclusion, or maintenance items]

NEXT SERVICE: [date]
Client Signature: _____________
```
