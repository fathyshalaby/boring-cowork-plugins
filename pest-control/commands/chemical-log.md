# Command: /pest:chemical-log

Generate a pesticide application record per regulatory requirements.

## Output Format
```
PESTICIDE APPLICATION RECORD
Date: [date] | Time: [start-end]
Technician: [name] | Cert #: [number] | State: [state]
Client: [name] | Address: [address]

APPLICATION DETAILS:
Target Pest: [species]
Product: [name] | EPA Reg #: [number]
Active Ingredient: [name] | Formulation: [type]
Dilution Rate: [X oz per gallon / ready-to-use]
Amount Used: [X oz/lbs/fl oz]
Application Method: [crack & crevice / spot / bait / etc.]
Areas Treated: [specific locations]

WEATHER (exterior): Temp: [X]°F | Wind: [X] mph | Precip: [Y/N]

SAFETY:
PPE Worn: [list]
Precautions Taken: [list]

Signature: _____________ Date: _________
```
