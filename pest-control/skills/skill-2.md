# Integrated Pest Management (IPM)
Apply the IPM framework for long-term, sustainable pest management:

### IPM Pyramid (Priority Order)
1. **Prevention/Exclusion** (most important): Seal entry points, repair screens, door sweeps, caulk gaps. Eliminate food/water/harborage.
2. **Cultural/Sanitation**: Clean up food debris, fix moisture issues, declutter, proper waste management.
3. **Mechanical/Physical**: Traps, monitors (glue boards, snap traps, pheromone traps), vacuuming, heat treatment.
4. **Biological**: Beneficial organisms (parasitic wasps for stored product pests, nematodes for grubs).
5. **Chemical** (last resort): Targeted application of least-toxic effective product. Baits before sprays.

### IPM Documentation
- Monitoring data: Trap maps, catch logs, inspection findings.
- Action thresholds: Define when treatment is warranted (e.g., >5 cockroaches on a glue board in 1 week).
- Treatment records: What was done, where, what product, how much, by whom.
- Follow-up: Verify treatment effectiveness at next service visit.

### IPM in Sensitive Environments
- **Schools (EPA School IPM)**: Notification requirements, restricted product lists, focus on non-chemical methods.
- **Healthcare facilities**: Joint Commission requires an IPM program. No broadcast spraying.
- **Food facilities**: GFSI/SQF/BRC all require IPM. Documentation is heavily audited.
