# Command: /actuary:guideline-draft

Generate an underwriting guideline based on actuarial analysis results.

## Output Format
```
UNDERWRITING GUIDELINE
Line: [name] | Effective: [date] | Approved By: [name]

SCOPE: [What risks this guideline applies to]

ACCEPTABLE RISKS:
[Criteria for risks that fall within appetite]
• Class: [acceptable classes/industries]
• Size: [premium or exposure range]
• Loss History: [acceptable loss ratio or frequency thresholds]
• Territory: [acceptable geographic areas]

REFERRAL REQUIRED:
[Risks requiring senior underwriter or actuarial review]
• [Criteria that trigger referral]

DECLINE:
[Risks outside appetite]
• [Hard decline criteria]

PRICING GUIDELINES:
[Rate ranges, minimum premiums, or pricing models to apply]

SUPPORTING DATA:
[Reference the actuarial analysis that supports these guidelines]
```
