# Reinsurance Analysis
Analyze reinsurance structures and their impact on net results:

### Common Reinsurance Structures
| Type | Description | Purpose |
|------|-----------|---------|
| Quota Share | Cede a fixed % of all premiums and losses | Capacity, surplus relief |
| Surplus Share | Cede amounts above a retained line | Large risk capacity |
| Per-Risk Excess | Reinsurer pays above retention per risk | Large loss protection |
| Catastrophe Excess | Reinsurer pays above retention per event | CAT loss protection |
| Aggregate Stop Loss | Reinsurer pays when total losses exceed threshold | Overall profitability protection |

### Analyzing Reinsurance Need
1. Calculate gross loss distribution (or use CAT model output).
2. Define the company's risk tolerance: maximum acceptable loss per event and in aggregate.
3. Size the retention: Balance premium ceded (cost) vs. protection provided (benefit).
4. Evaluate reinsurance pricing: Compare reinsurance premium to the expected loss transfer. The ratio is the "rate on line."

### Reinsurance Impact on Financials
- **Net written premium**: Gross premium - ceded premium.
- **Net incurred losses**: Gross losses - recoveries from reinsurers.
- **Ceding commission**: Under quota share treaties, the reinsurer pays a commission to cover the cedant's acquisition costs. This can significantly improve the cedant's expense ratio.
- Reinsurance recoverables are balance sheet assets—monitor reinsurer credit quality.
