# Command: /waste:tonnage-report

Generate a monthly tonnage and diversion report for internal or client use.

## Output Format
```
MONTHLY TONNAGE & DIVERSION REPORT
Period: [Month Year]
Service Area: [name/zone]

COLLECTION SUMMARY:
Stream              Tons     % of Total    vs. Last Month
─────────────────────────────────────────────────────────
Municipal SW        [X]      [X]%          [+/- X]%
Recycling           [X]      [X]%          [+/- X]%
Organics/Compost    [X]      [X]%          [+/- X]%
C&D                 [X]      [X]%          [+/- X]%
Hazardous           [X]      [X]%          [+/- X]%

TOTAL COLLECTED: [X] tons
TOTAL DIVERTED:  [X] tons
DIVERSION RATE:  [X]% (Target: [X]%)

DISPOSAL DESTINATIONS:
Facility            Tons     Tipping Fee    Total Cost
─────────────────────────────────────────────────────
[Landfill name]     [X]      $[X]/ton       $[X]
[MRF name]          [X]      $[X]/ton       $[X]
[Composting]        [X]      $[X]/ton       $[X]

YEAR-TO-DATE:
Total Collected: [X] tons | Diverted: [X] tons | Rate: [X]%
```
