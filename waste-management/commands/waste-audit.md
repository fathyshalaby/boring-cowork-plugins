# Command: /waste:waste-audit

Generate a waste characterization audit report for a commercial client.

## Output Format
```
WASTE CHARACTERIZATION AUDIT REPORT
Client: [name] | Location: [address]
Audit Date: [date] | Conducted By: [name]

METHODOLOGY:
[X] containers were sampled and sorted by hand into [X] material categories.

COMPOSITION RESULTS:
Material              Weight (lbs)    % of Total    Divertible?
───────────────────────────────────────────────────────────────
Mixed Paper/Cardboard   [X]            [X]%          YES
Plastics (#1-7)         [X]            [X]%          YES
Glass                   [X]            [X]%          YES
Metals                  [X]            [X]%          YES
Food Waste              [X]            [X]%          YES (compost)
Non-Recyclable          [X]            [X]%          NO
Contamination           [X]            [X]%          N/A

TOTAL SAMPLE: [X] lbs
CURRENT DIVERSION RATE: [X]%
POTENTIAL DIVERSION RATE: [X]% (with recommended improvements)

RECOMMENDATIONS:
1. [Specific action to improve diversion]
2. [Specific action]
3. [Specific action]

ESTIMATED ANNUAL SAVINGS: $[X] (through right-sizing and diversion)
```
