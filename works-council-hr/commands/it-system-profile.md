# Command: /hr-wc:it-system-profile

Generate an IT system description for co-determination negotiation (monitoring/privacy).

## Output Format
```
IT SYSTEM DEPLOYMENT PROFILE (Works Council Notification)
System Name: [e.g., Salesforce / Microsoft Teams / Internal App]
Vendor: [Name] | Hosting: [Cloud/On-Premise, Geographic location of servers]

1. PURPOSE OF IMPLEMENTATION
[Business benefit, e.g., streamlining customer data, enabling remote communication]

2. IMPACT ON EMPLOYEES / USER GROUPS
[Which departments will use it? Is usage mandatory?]

3. DATA PROCESSED (Data Categories)
- Master data: [Name, email, role]
- Usage/Log data: [Sign-in times, IP address, device info]
- Content data: [Messages, files uploaded, CRM entries]

4. PERFORMANCE AND BEHAVIOR MONITORING POTENTIAL
[Crucial section for WC approval]
Does the system generate automated reports on individual productivity? [Yes/No]
Are analytics activated? [e.g., Viva Insights]
*Employer formal commitment*: Management guarantees that log files and usage metrics from this system will NOT be used for continuous individual performance control or disciplinary measures, except in cases of suspected criminal activity or severe policy violation.

5. ACCESS RIGHTS & ROLES
[Who can see what? Admin vs Manager vs Basic User]

6. DATA RETENTION
[How long is data kept before automatic deletion?]
```
