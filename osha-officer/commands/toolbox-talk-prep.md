# Command: /osha:toolbox-talk-prep

Generate a toolbox talk outline for a safety topic.

## Output Format
```
TOOLBOX TALK
Topic: [subject] | Date: [date]
Presenter: [name] | Duration: [X] minutes

WHY THIS MATTERS:
[Brief statement connecting to workplace relevance]

KEY POINTS:
1. [Main point with specific guidance]
2. [Second point]
3. [Third point]

DISCUSSION QUESTIONS:
• [Question to engage workers]
• [Scenario-based question]

ACTION ITEM:
[One specific takeaway for today's work]

ATTENDANCE:
Name                    Signature
──────────────────────────────────
____________________    __________
```
