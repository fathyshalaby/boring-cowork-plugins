# Command: /osha:citation-response

Generate a framework for responding to an OSHA citation.

## Output Format
```
OSHA CITATION RESPONSE
Inspection #: [number] | Citation #: [number]
Date Issued: [date] | 15-Day Deadline: [date]
Standard Cited: [29 CFR X.X]
Classification: [Serious/Other/Willful/Repeat]
Proposed Penalty: $[X]

ALLEGED VIOLATION:
[Summary of OSHA's citation language]

RESPONSE STRATEGY:
☐ Accept and abate
☐ Informal conference (request by [date])
☐ Contest (file Notice of Contest by [date])

ABATEMENT PLAN:
Action                          Owner      Due Date
──────────────────────────────────────────────────────
[corrective action]             [name]     [date]

ABATEMENT DOCUMENTATION TO SUBMIT:
☐ Description of corrective action taken
☐ Date of correction
☐ Photos/evidence of abatement
☐ Training records (if applicable)
```
