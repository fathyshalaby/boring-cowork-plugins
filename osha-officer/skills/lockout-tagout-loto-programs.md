# Lockout/Tagout (LOTO) Programs
Implement the Control of Hazardous Energy program:

### LOTO Requirements (1910.147)
- Written energy control procedures for each machine/equipment.
- Employee training: authorized (performs LOTO), affected (works near), other (works in area).
- Periodic inspection of energy control procedures at least annually.
- Locks and tags provided by the employer, standardized.

### Energy Control Procedure
1. **Notify affected employees** that the machine will be locked out.
2. **Shut down** the machine using normal stopping procedure.
3. **Isolate** from all energy sources (electrical, mechanical, hydraulic, pneumatic, chemical, thermal, gravity).
4. **Apply locks and tags** at each energy isolation device.
5. **Release stored energy** (bleed hydraulic/pneumatic lines, block elevated parts, discharge capacitors).
6. **Verify isolation** — attempt to start the machine. Return controls to off/neutral.

### Group LOTO
- When multiple workers service the same machine, each authorized employee applies their own lock.
- Use a lockout hasp to allow multiple locks on one energy isolation point.
- Designate a primary authorized employee to coordinate group LOTO.

### Common Violations
- No written energy control procedures.
- Failure to perform periodic inspections.
- Using tags only (without locks) when locks are possible.
- Not accounting for all energy sources (gravity, stored pressure, thermal).
