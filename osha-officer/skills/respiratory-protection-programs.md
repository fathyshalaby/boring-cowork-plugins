# Respiratory Protection Programs
Implement respiratory protection per OSHA 1910.134:

### Program Requirements
1. **Written program**: Worksite-specific, administered by a program administrator with appropriate training.
2. **Medical evaluation**: Before fit testing, employee must complete OSHA medical questionnaire. PLHCP determines medical clearance.
3. **Fit testing**: Annual for tight-fitting respirators. Qualitative (pass/fail taste/smell test) or quantitative (instrument measures leakage).
4. **Training**: How to use, limitations, maintenance, and inspection of respirators.
5. **Maintenance**: Inspection before each use, cleaning after each use, proper storage.

### Respirator Selection
| Hazard | Respirator Type |
|--------|----------------|
| Nuisance dust (< PEL) | Filtering facepiece (N95) |
| Particulates (above PEL) | Half-face APR with P100 filters |
| Organic vapors/gases | Half/full-face APR with appropriate cartridges |
| IDLH atmospheres | SCBA or supplied air with escape bottle |
| Oxygen deficient (<19.5%) | SCBA or supplied air |
| Unknown atmospheres | SCBA only |

### Voluntary Use
- If employees choose to wear filtering facepieces (N95) voluntarily, provide Appendix D (information on voluntary use).
- No fit test or medical evaluation required for voluntary filtering facepiece use.
- For voluntary use of elastomeric respirators, medical evaluation IS required.
