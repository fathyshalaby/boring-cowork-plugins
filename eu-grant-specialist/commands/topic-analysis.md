# Command: /eu-grants:topic-analysis

Analyze a Funding & Tenders portal Call Topic extract to structure a proposal response.

## Output Format
```
EU CALL TOPIC ANALYSIS
Topic ID: [e.g., HORIZON-CL5-2024-D2-01-01]
Topic Title: [Title]
Budget per project: [Expected EU contribution, e.g., €3-5M]
Action Type: [e.g., RIA, IA, CSA]

EXPECTED OUTCOMES (Must Address):
The proposal should contribute to the following outcomes:
☐ [Outcome 1] -> Proposed Approach: [Short strategy]
☐ [Outcome 2] -> Proposed Approach: [Short strategy]

SCOPE REQUIREMENTS (Must Implement):
☐ [Scope element 1] -> Covered in WP: [X]
☐ [Scope element 2] -> Covered in WP: [X]
☐ Additional requirements: [E.g., SSH integration, TRL starting/ending levels]

COMPETITIVE EDGE STRATEGY:
[What will make this proposal stand out against others addressing the same text? E.g., specific end-user involvement, unique methodology]

CONSORTIUM NEEDS:
Required expertise to cover the scope:
1. [Expertise/Role] (Filled by: [Partner?])
2. [Expertise/Role] (Need to recruit)
```
