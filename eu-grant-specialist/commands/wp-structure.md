# Command: /eu-grants:wp-structure

Generate the structure for a specific Work Package.

## Output Format
```
WP[X]: [Work Package Title]
Lead Beneficiary: [Partner Name]
Start M[X] - End M[X] | Total Person Months: [X]

OBJECTIVES:
• [Clear, actionable objective 1]
• [Clear, actionable objective 2]

TASKS:
Task X.1: [Task Title] (Lead: [Partner], M[X]-M[Y])
[Brief description of methodology and partner roles]

Task X.2: [Task Title] (Lead: [Partner], M[X]-M[Y])
[Brief description of methodology and partner roles]

DELIVERABLES:
Dx.1: [Title] (Type: [Report/Demo/Data/Other], Dissemination: [PU/SEN], Due: M[X])
Dx.2: [Title] (Type: [Report/Demo/Data/Other], Dissemination: [PU/SEN], Due: M[X])

MILESTONES (Decision points/Control points):
Mx.1: [Title] (Due: M[X], Means of Verification: [e.g., Successful prototype test report submitted])
```
