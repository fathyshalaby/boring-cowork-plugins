# Quality & Receiving Inspection
Ensure incoming parts meet quality requirements:

### Receiving Inspection Process
1. Verify PO match: part number, revision, quantity, and description.
2. Visual inspection: packaging damage, corrosion, contamination.
3. Documentation: Certificate of Conformance (COC), test reports, material certs (if required).
4. Dimensional/functional check per sampling plan (for critical parts).
5. Accept or reject. If rejected: quarantine, tag, and initiate Discrepancy Report.

### Inspection Levels
| Part Type | Inspection Level |
|-----------|------------------|
| Critical/safety parts | 100% inspection or statistical sampling (AQL 1.0) |
| Standard production parts | Skip-lot after qualifying 3 consecutive good lots |
| MRO/consumables | Visual + documentation check |
| Calibrated instruments | Full calibration verification |

### Non-Conformance Management
- **Discrepancy Report (DR)**: Document the non-conformance: what was ordered vs. what was received.
- **Disposition**: Return to vendor, use as-is (with engineering approval/waiver), scrap, or rework.
- **Vendor notification**: Issue SCAR for significant non-conformances.
- **Cost recovery**: Debit memo for defective materials, freight for returns.
