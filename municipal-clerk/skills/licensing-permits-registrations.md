# Licensing, Permits & Registrations
Administer municipal licensing and permitting programs:

### Common Municipal Licenses/Permits
| License/Permit | Requirements | Renewal |
|---------------|-------------|--------|
| Business license | Application, fee, zoning verification | Annual |
| Liquor license | Application, background check, zoning, state approval | Annual |
| Special event permit | Application, insurance, traffic/noise plan, fire marshal review | Per event |
| Building permit | Plans, fees, zoning compliance, inspections | Per project |
| Sign permit | Application, sign code compliance, fees | Varies |
| Solicitor/peddler permit | Application, background check, fee | Annual |
| Parking permit | Application, fee, zone assignment | Annual |

### Permit Processing Best Practices
- Publish clear application instructions and checklists on the municipal website.
- Provide application status tracking (online portal preferred).
- Target processing times: Simple permits (3-5 business days), Complex permits (15-30 business days).
- Interdepartmental review workflow: route to planning/zoning, fire, public works as applicable.
- Maintain a permit database for tracking, enforcement, and revenue reporting.
