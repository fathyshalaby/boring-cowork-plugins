# Ordinance & Resolution Drafting
Draft, codify, and maintain municipal ordinances and resolutions:

### Ordinances vs. Resolutions
| | Ordinance | Resolution |
|---|-----------|------------|
| Legal weight | Law — binding and enforceable | Policy statement — administrative action |
| Adoption | Typically requires two readings and a public hearing | Usually adopted at one meeting |
| Publication | Must be published per state law | Publication usually not required |
| Codification | Added to the municipal code | Not codified |
| Examples | Zoning regulations, tax levies, traffic regulations | Budget adoption, appointments, commendations |

### Ordinance Drafting Standards
- Title clearly states the subject matter.
- Whereas clauses (recitals) state the purpose and authority.
- Enacting clause: "Be it ordained by the [Governing Body] of the [Municipality]..."
- Body: numbered sections with clear, enforceable language.
- Penalty clause for violations (if applicable).
- Effective date and publication requirements.
- Amend existing code sections by reference — do not create standalone conflicting provisions.

### Codification
- Maintain an organized municipal code (Municode, American Legal Publishing, or in-house).
- Supplement the code after each ordinance adoption.
- Regular code review to identify outdated, conflicting, or unenforceable provisions.
