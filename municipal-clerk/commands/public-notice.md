# Command: /clerk:public-notice

Generate a public notice for newspaper publication.

## Output Format
```
PUBLIC NOTICE
[MUNICIPALITY NAME]

[NOTICE TYPE: e.g., Notice of Public Hearing, Notice of Ordinance, etc.]

[Body of notice — meets all statutory requirements for content]

Date: [date of hearing/effective date/etc.]
Time: [time]
Location: [address]

[Contact information for questions]

[Clerk Name]
[Title]
Publish: [publication date(s)]
```
