# Command: /clerk:zoning-check

Generate a zoning verification letter for a property.

## Output Format
```
ZONING VERIFICATION LETTER
Date: [date]
Property Address: [address]
Parcel ID: [number]
Requested By: [name]

ZONING CLASSIFICATION: [zone code and description]
COMPREHENSIVE PLAN: [future land use designation]

PERMITTED USES: [list principal uses]
CONDITIONAL USES: [list uses requiring special approval]

SETBACKS: Front: [X]' | Side: [X]' | Rear: [X]'
MAX HEIGHT: [X]' | LOT COVERAGE: [X]%

OVERLAY DISTRICTS: [if applicable]
SPECIAL CONDITIONS: [if applicable]

This letter is for informational purposes only and does not
constitute approval for any specific use or development.

[Clerk/Zoning Administrator Name]
[Title]
```
