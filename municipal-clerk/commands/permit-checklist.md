# Command: /clerk:permit-checklist

Generate a permit application completeness checklist.

## Output Format
```
PERMIT APPLICATION CHECKLIST
Permit Type: [type] | Date Received: [date]
Applicant: [name] | Property: [address]

REQUIRED DOCUMENTS:
☐ Completed application form
☐ Application fee ($[X]) ☐ Received
☐ [Document specific to permit type]
☐ [Additional requirements]

DEPARTMENT REVIEWS:
☐ Planning/Zoning: [status]
☐ Building: [status]
☐ Fire: [status]
☐ Public Works: [status]

APPLICATION STATUS: ☐ Complete ☐ Incomplete
MISSING ITEMS: [list]
NOTIFIED APPLICANT: [date/method]
```
