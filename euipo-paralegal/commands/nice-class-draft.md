# Command: /euipo:nice-class-draft

Draft a compliant specification of goods/services for an EUTM application.

## Output Format
```
NICE CLASSIFICATION DRAFTING
Core Product/Business: [e.g., A mobile app for booking fitness classes and selling branded yoga mats]

CLASS 9 (Software)
- Avoid: "Computer software" (Too broad, likely face clarity objection).
- Preferred HDB Terms: Downloadable application software for smart phones; Application software for mobile devices related to fitness and health; Downloadable software to enable the booking of sports facilities.

CLASS 27 (Sporting Goods)
- Preferred HDB Terms: Yoga mats; Gymnastic mats; Personal exercise mats.

CLASS 41 (Education/Entertainment)
- Preferred HDB Terms: Arranging and conducting of fitness classes; Health club services [health and fitness training]; Providing online information relating to fitness.

FAST TRACK COMPLIANCE
- All listed terms exactly match the EUIPO Harmonised Database (HDB).
- Classification qualifies for Fast Track examination (reducing examination time by half).
```\n
