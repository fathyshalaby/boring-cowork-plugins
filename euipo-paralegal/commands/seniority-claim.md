# Command: /euipo:seniority-claim

Determine if a National Mark can be claimed as Seniority for an EUTM.

## Output Format
```
SENIORITY CLAIM ASSESSMENT

NATIONAL MARK (Prior Right)
- Registry: [e.g., INPI (France)]
- Mark: ["LUMINA" (Word mark)]
- Filing Date: [01 Jan 2015]
- Owner: [Lumina Corp Ltd.]
- Goods: [Class 11: Lighting installations; lamps]

EUTM APPLICATION
- Mark: ["Lumina" (Word mark) - Note: Case difference is acceptable for word marks].
- Owner: [Lumina Corp Limited] -> REQUIRES RECORDAL. (Names must match exactly. Update French register to "Limited" before claiming).
- Goods Claimed in EUTM: [Class 11: Apparatus for lighting; LED luminaires; lamps].

TRIPLE IDENTITY ANALYSIS
1. Identical Mark? [YES - Both are word marks of the identical string].
2. Identical Owner? [NO - Administrative mismatch. Action required before filing].
3. Identical/Narrower Goods? [PARTIAL - "Lamps" is identical. However, the EUTM claims broader terms "Apparatus for lighting". Seniority can only be claimed for the exactly overlapping term "lamps"].

RECOMMENDATION
File recordal for name change in France. Once confirmed, file EUTM but restrict the seniority claim explicitly to "lamps" in Class 11 to avoid examiner objection.
```\n
