# Command: /docs:tutorial-draft

Generate a tutorial structure for a feature or task.

## Output Format
```
# Tutorial: [Title]

## Overview
[What the user will learn and build]

## Prerequisites
- [Tool/account/knowledge needed]

## Steps

### Step 1: [Action verb + object]
[Instructions with code blocks]

### Step 2: [Action verb + object]
[Instructions with code blocks]

[Continue for all steps]

## Verify
[How to confirm it worked]

## Next Steps
[What to learn/do next with links]

## Troubleshooting
[Common issues and solutions]
```
