# Command: /docs:release-notes

Generate formatted release notes from a list of changes.

## Output Format
```
# Release v[X.Y.Z] — [Date]

## Highlights
[1-2 sentence summary]

## ✨ New Features
- **[Feature]**: [User-facing description]

## 🔧 Improvements
- **[Area]**: [What changed, user benefit]

## 🐛 Bug Fixes
- Fixed [specific issue] (#[number])

## ⚠️ Breaking Changes
- [Change description + migration path]

## 📋 Full Changelog
[link to diff or PR list]
```
