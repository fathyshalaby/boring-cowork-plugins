# Command: /docs:openapi-to-markdown

Convert an OpenAPI specification to readable Markdown documentation.

## Output Format
```
# API Reference

## Authentication
[Auth method and usage]

## Endpoints

### [Group Name]

#### METHOD /path
[Description]

**Parameters:**
| Name | In | Type | Required | Description |

**Request Body:**
```json
[example]
```

**Response 200:**
```json
[example]
```

**Errors:**
| Status | Description |
```
