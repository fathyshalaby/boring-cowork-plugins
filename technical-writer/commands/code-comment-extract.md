# Command: /docs:code-comment-extract

Extract and organize inline code documentation into a reference document.

## Output Format
```
# [Module/File] Reference

## Functions

### function_name(params)
[Extracted docstring/JSDoc]

**Parameters:**
| Name | Type | Description |

**Returns:** [type — description]

**Example:**
```
[usage example]
```

[Continue for all documented functions/classes]

## Undocumented Items
[List of functions/classes missing documentation]
```
