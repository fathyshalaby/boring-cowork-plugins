# Command: /docs:changelog-update

Generate a Keep a Changelog formatted entry.

## Output Format
```
## [Unreleased]

### Added
- [New feature description]

### Changed
- [Changed behavior description]

### Deprecated
- [Feature deprecated, alternative noted]

### Removed
- [Feature removed]

### Fixed
- [Bug fix description] (#[issue])

### Security
- [Security fix description]
```
