# Command: /patent:drawing-desc

Generate brief description of drawings for the specification.

## Output Format
```
BRIEF DESCRIPTION OF DRAWINGS

FIG. 1 is [a perspective view / a block diagram / a flowchart]
showing [description].

FIG. 2 is [a cross-sectional view / a schematic diagram]
illustrating [description].

[Continue for each figure]

DETAILED REFERENCE NUMERALS:
Ref#    Description
─────────────────────────────
[100]   [element name]
[102]   [element name]
[200]   [element name]
```
