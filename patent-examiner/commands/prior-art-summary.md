# Command: /patent:prior-art-summary

Generate a prior art search summary report.

## Output Format
```
PRIOR ART SEARCH REPORT
Invention: [description]
Search Date: [date] | Searcher: [name]

KEY CLAIMS ANALYZED: [summary of novelty features]

DATABASES SEARCHED:
[List databases, classification codes, and keyword strategies]

RELEVANT REFERENCES:
#  Ref#          Date     Relevance    Key Disclosure
────────────────────────────────────────────────────────────
1  US [number]   [date]   §102/§103    [what it teaches]
2  [publication]  [date]   §103         [what it teaches]

ANALYSIS:
[How the references relate to the claimed invention]

CONCLUSION:
☐ Novel and non-obvious  ☐ Anticipation risk  ☐ Obviousness risk
```
