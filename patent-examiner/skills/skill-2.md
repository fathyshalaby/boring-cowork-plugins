# Prior Art Search Strategies
Conduct comprehensive prior art searches:

### Search Resources
| Resource | Best For |
|----------|----------|
| USPTO PatFT/AppFT | US patents and published applications |
| Google Patents | Broad search, full-text, multi-jurisdiction |
| Espacenet | European and international patents (EPO) |
| WIPO PatentScope | PCT international applications |
| IEEE Xplore, PubMed | Non-patent literature (NPL) |
| Google Scholar | Academic papers, technical publications |

### Search Methodology
1. **Understand the invention**: Read the specification and claims thoroughly. Identify the novel features.
2. **Classification search**: Find relevant CPC/IPC classifications. Search within those classes.
3. **Keyword search**: Use synonyms, alternative terms, and Boolean operators. Inventors use different terminology than the searcher might expect.
4. **Citation analysis**: Review references cited by and citing relevant patents (forward and backward citation chains).
5. **Inventor/assignee search**: Search for other filings by the same inventor or assignee.
6. **NPL search**: Search academic databases, technical publications, product catalogs.

### Search Documentation
- Record all search queries, databases searched, date of search, and results reviewed.
- For each relevant reference, note: publication number/date, relevant disclosure, and which claim elements are taught.
- Prioritize references by relevance: anticipatory (§102), combinable for obviousness (§103), or relevant background.
