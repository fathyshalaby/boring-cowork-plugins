# Multi-Point Vehicle Inspection
Perform thorough multi-point inspections on every vehicle that enters the shop:

### Inspection Checklist
**Under Hood**:
- [ ] Engine oil level and condition (clean amber → dark → black/gritty)
- [ ] Coolant level and condition (use a refractometer for freeze point)
- [ ] Brake fluid level and color (clear/amber = good; dark brown = flush needed)
- [ ] Power steering fluid level
- [ ] Transmission fluid level and color (ATF: red/pink = good; brown = service needed)
- [ ] Drive belts: cracking, fraying, glazing
- [ ] Hoses: soft spots, bulging, leaks
- [ ] Battery: terminal corrosion, hold-down bracket, load test result
- [ ] Air filter condition

**Under Vehicle**:
- [ ] Brake pad/shoe thickness (specify mm for each axle)
- [ ] Rotor condition: thickness, scoring, warping
- [ ] Tire tread depth (measure at 3 points across the tread, all 4 tires + spare)
- [ ] Tire pressure (check against door placard specification)
- [ ] CV boots/axle condition
- [ ] Exhaust: leaks, rust-through, hanger condition
- [ ] Suspension: ball joints, tie rod ends, struts/shocks (bounce test)
- [ ] Fluid leaks: oil, coolant, trans, power steering, differential

**Interior/Electrical**:
- [ ] All lights: headlights, taillights, turn signals, brake lights
- [ ] Wiper blades: streak-free or replacement needed
- [ ] Horn operation
- [ ] Window operation
- [ ] HVAC operation (heat and AC)

### Color-Coded Reporting
Use a traffic light system for customer clarity:
- 🟢 **Green**: Component is in good condition. No action needed.
- 🟡 **Yellow**: Component is showing wear. Plan for service soon.
- 🔴 **Red**: Component needs immediate attention (safety or reliability risk).
