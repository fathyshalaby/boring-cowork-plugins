# Scheduling & Room Assignment
Optimize course scheduling and room utilization:

### Scheduling Principles
- **Demand-driven**: Schedule sections based on enrollment projections, not tradition.
- **Room utilization target**: 65-75% seat occupancy, 60-70% time utilization is considered efficient.
- **Standard meeting patterns**: MWF, TR, MW, etc. Standardized time blocks maximize room flexibility.
- **Finals scheduling**: Map meeting patterns to final exam slots. Avoid conflicts for students.

### Room Assignment Optimization
| Factor | Consideration |
|--------|---------------|
| Capacity | Match enrollment cap to room size (avoid 15 students in a 100-seat room) |
| Technology | Smart classroom needs, computer labs, recording capability |
| Accessibility | ADA-compliant room when needed |
| Proximity | Back-to-back classes need rooms in the same building or nearby |
| Special needs | Labs, studios, performance spaces |

### Scheduling Software
- **Ad Astra, 25Live, CollegeScheduler**: Automated scheduling optimization.
- Event scheduling (25Live): Room reservations for non-academic events.
- Integration with SIS (Student Information System) for enrollment data.
