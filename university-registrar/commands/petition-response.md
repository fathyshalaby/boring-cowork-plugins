# Command: /registrar:petition-response

Generate a response to a student academic petition.

## Output Format
```
[Date]
[Student Name and Address]

Re: Academic Petition — [Type: Late Withdrawal / Grade Appeal /
    Policy Exception / Readmission]
    Student ID: [number]

Dear [Student Name],

The [Committee Name] has reviewed your petition dated [date]
regarding [brief description of the request].

DECISION: ☐ APPROVED  ☐ DENIED  ☐ APPROVED WITH CONDITIONS

[If approved]: [Description of action taken and any conditions]
[If denied]: [Rationale for denial based on policy]

APPEAL PROCESS:
[Information about further appeal options, if any]

Sincerely,
[Name, Title]
Office of the Registrar
```
