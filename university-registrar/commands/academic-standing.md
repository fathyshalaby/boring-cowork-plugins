# Command: /registrar:academic-standing

Generate an academic standing report for end-of-term processing.

## Output Format
```
ACADEMIC STANDING REPORT
Term: [term] | Run Date: [date]

SUMMARY:
Total Students Evaluated: [X]
Good Standing: [X] ([X]%)
Probation (new): [X]
Probation (continued): [X]
Suspension: [X]
Dean's List: [X]
President's List: [X]

NOTIFICATION STATUS:
☐ Probation letters sent: [date]
☐ Suspension letters sent: [date]
☐ Dean's List letters sent: [date]
☐ Advisor notifications sent: [date]

APPEAL DEADLINE: [date]
```
