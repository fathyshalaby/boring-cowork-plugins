# Command: /registrar:degree-audit

Generate a degree audit summary for a student.

## Output Format
```
DEGREE AUDIT SUMMARY
Student: [name] | ID: [number]
Program: [degree, major] | Catalog Year: [year]
Credits Completed: [X] | Credits Remaining: [X]
GPA: [X] | Expected Graduation: [term]

REQUIREMENT STATUS:
Category                  Required  Completed  Remaining
──────────────────────────────────────────────────────────
General Education          [X]       [X]        [X]
Major Core                 [X]       [X]        [X]
Major Electives            [X]       [X]        [X]
Free Electives             [X]       [X]        [X]
Total Credits              [X]       [X]        [X]

OUTSTANDING REQUIREMENTS:
• [Specific courses or categories still needed]

NOTES: [substitutions, exceptions, advisor comments]
```
