# Command: /infosec:policy-draft

Generate a security policy document.

## Output Format
```
[ORGANIZATION NAME]
INFORMATION SECURITY POLICY: [Topic]

Version: [X.X] | Effective: [date]
Owner: [name/title] | Approved By: [name/title]

1. PURPOSE
[Why this policy exists]

2. SCOPE
[Who and what this policy applies to]

3. POLICY STATEMENTS
3.1 [Specific requirement]
3.2 [Specific requirement]
[etc.]

4. ROLES AND RESPONSIBILITIES
[Who is responsible for what]

5. ENFORCEMENT
[Consequences of non-compliance]

6. EXCEPTIONS
[Process for requesting policy exceptions]

7. RELATED DOCUMENTS
[References to related policies, standards, procedures]

8. REVISION HISTORY
Version    Date    Author    Changes
──────────────────────────────────────
[X.X]      [date]  [name]   [summary]
```
