# Command: /realestate:listing-description

Generate a compelling, professional MLS listing description for a property.

## Required Input
- Property address, price, beds/baths/SF
- Key features and upgrades
- Neighborhood highlights
- Agent name and contact

## Output Format
Generate a 150-300 word description that:
1. Opens with the most compelling feature or lifestyle hook
2. Flows room-by-room through key selling points
3. Highlights recent upgrades and unique features
4. Describes the neighborhood and location advantages
5. Closes with a call to action

**Rules:**
- Do NOT use all caps, excessive exclamation points, or superlatives.
- Do NOT violate Fair Housing (describe the property, not the people).
- Use vivid, specific language over generic ("chef's kitchen" vs. "nice kitchen").
- Include key search terms naturally for online discoverability.
