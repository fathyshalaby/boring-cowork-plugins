# Command: /realestate:cma-report

Generate a structured CMA report for a listing presentation or buyer consultation.

## Output Format
```
COMPARATIVE MARKET ANALYSIS
Subject Property: [Address]
Prepared By: [Agent] | Date: [date]

SUBJECT PROPERTY DETAILS:
[beds/baths/SF/lot/year built/features]

RECENTLY SOLD COMPARABLES:
Comp  Address         Sold Price  $/SF    Beds/Bath  SF     DOM  Sold Date
─────────────────────────────────────────────────────────────────────────
1     [address]       $[X]        $[X]    [X/X]      [X]    [X]  [date]
2     [address]       $[X]        $[X]    [X/X]      [X]    [X]  [date]
3     [address]       $[X]        $[X]    [X/X]      [X]    [X]  [date]

ADJUSTMENTS:
[Brief explanation of any adjustments made for differences]

ACTIVE LISTINGS (Competition):
[Currently listed comps with asking prices]

SUGGESTED LIST PRICE RANGE: $[X] - $[X]
RECOMMENDED LIST PRICE: $[X]

MARKET CONDITIONS:
Average DOM: [X] days | Months of Inventory: [X]
Median $/SF: $[X] | List-to-Sale Ratio: [X]%
```
