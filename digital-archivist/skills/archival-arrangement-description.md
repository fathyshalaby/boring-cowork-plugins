# Archival Arrangement & Description
Organize archival collections following professional standards:

### Levels of Description
| Level | Scope | Typical Size |
|-------|-------|--------------|
| Repository | Entire institution | All holdings |
| Collection/Fonds | All records from one creator | Varies |
| Series | Records grouped by activity/form | Typically boxes |
| File/Folder | Documents grouped within a series | Single folder |
| Item | Individual document or object | Single item |

### DACS (Describing Archives: A Content Standard)
- Required elements: Title, Date, Extent, Creator, Scope & Content, Conditions Governing Access, Languages.
- Describe at the lowest level that is practical and useful. Not every item needs item-level description.
- **Respect des fonds**: Maintain the original order and provenance of records. Do NOT rearrange materials into subject categories.

### Finding Aid Structure
1. Title page and summary
2. Administrative information (provenance, access restrictions, citation)
3. Biographical/Historical note
4. Scope and content note
5. Series descriptions
6. Container list (box/folder inventory)
