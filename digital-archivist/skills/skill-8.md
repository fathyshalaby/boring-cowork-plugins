# Digital Exhibits & Public Access
Create engaging digital exhibits and public access points:

### Exhibit Planning
1. Define the audience and narrative theme.
2. Select materials from the collection that tell the story.
3. Write interpretive text (accessible language, 8th grade reading level).
4. Design the exhibit layout (chronological, thematic, geographic).
5. Build in the platform (Omeka, WordPress, custom).
6. Accessibility: Alt text for all images, captions for audio/video, keyboard navigation.

### SEO for Digital Collections
- Descriptive page titles and meta descriptions.
- Structured data (Schema.org) for cultural heritage objects.
- Open Graph tags for social media sharing.
- Sitemaps submitted to search engines.
- Persistent identifiers (DOIs, Handles, ARKs) for stable URLs.

### User Engagement
- Crowdsourcing: Invite the public to transcribe handwritten documents or identify photographs.
- Social media: Share "this day in history" posts featuring collection items.
- Educational resources: Create lesson plans and primary source sets for teachers.
