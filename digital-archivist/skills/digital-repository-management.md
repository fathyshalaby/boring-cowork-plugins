# Digital Repository Management
Manage digital repository systems for access and preservation:

### Common Repository Platforms
| Platform | Type | Best For |
|----------|------|----------|
| ArchivesSpace | Finding aid management | Archival description |
| DSpace | Institutional repository | Academic publications, datasets |
| Islandora | Fedora-based DAMS | Complex digital objects |
| Omeka | Exhibition/collection platform | Public-facing digital exhibits |
| Preservica | Digital preservation | Long-term preservation |
| Archivematica | Preservation processing | OAIS-compliant ingest |

### OAIS Reference Model
The Open Archival Information System is the standard framework:
- **SIP** (Submission Information Package): What comes in from the producer.
- **AIP** (Archival Information Package): What's stored in the repository.
- **DIP** (Dissemination Information Package): What's delivered to users.

### Access Management
- Define access levels: Public, restricted (on-site only), embargoed, dark archive.
- Implement rights statements (RightsStatements.org) for each object.
- Track usage metrics: downloads, page views, unique visitors. Demonstrate value to stakeholders.
