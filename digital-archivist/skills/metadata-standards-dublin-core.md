# Metadata Standards & Dublin Core
Apply metadata standards for digital preservation and discovery:

### Dublin Core Elements
| Element | Definition | Example |
|---------|-----------|--------|
| Title | Name of the resource | "Annual Report 1952" |
| Creator | Entity primarily responsible | "Smith, John" |
| Subject | Topic keywords | "Agriculture--Kansas--History" |
| Description | Free-text summary | "Annual report of farming activities..." |
| Publisher | Entity responsible for making available | "Kansas State Historical Society" |
| Date | Date of creation or publication | "1952-03-15" (ISO 8601) |
| Type | Nature of the resource | "Text", "Image", "Sound" |
| Format | File format | "application/pdf", "image/tiff" |
| Identifier | Unique identifier | "KHS-AR-1952-001" |
| Language | Language of content | "eng" (ISO 639-3) |
| Rights | Rights/access information | "Public Domain" |

### Controlled Vocabularies
- **LCSH** (Library of Congress Subject Headings): Standard subject terms.
- **AAT** (Art & Architecture Thesaurus): Terms for art, architecture, and material culture.
- **TGM** (Thesaurus for Graphic Materials): Subject terms for images.
- **LCNAF** (Library of Congress Name Authority File): Authorized forms of personal and corporate names.

Consistency is everything. Use controlled vocabularies rather than free text whenever possible.
