# Command: /archive:finding-aid

Generate a DACS-compliant finding aid structure for an archival collection.

## Output Format
```
FINDING AID
Collection: [title]
Dates: [inclusive dates]
Extent: [linear feet / number of items]
Creator: [name]
Repository: [institution]
Call Number: [number]

BIOGRAPHICAL/HISTORICAL NOTE:
[Context about the creator]

SCOPE AND CONTENT:
[What the collection contains and its research value]

ARRANGEMENT:
[How the collection is organized]

CONTAINER LIST:
Box  Folder  Title                    Dates
──────────────────────────────────────────────
1    1       [folder title]           [dates]
1    2       [folder title]           [dates]
```
