# Command: /archive:dublin-core

Generate Dublin Core metadata records for digital objects.

## Output Format
```
DUBLIN CORE METADATA RECORD
Identifier: [unique ID]
Title: [title]
Creator: [name]
Date: [ISO 8601 date]
Subject: [LCSH terms]
Description: [free-text summary]
Type: [DCMI type]
Format: [MIME type]
Language: [ISO 639-3]
Rights: [rights statement]
Relation: [related resources]
Coverage: [temporal/spatial]
Publisher: [publisher]
Contributor: [other contributors]
Source: [original source]
```
