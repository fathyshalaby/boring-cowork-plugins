# Client Contract Management
Manage commercial cleaning contracts professionally:

### Contract Types
| Type | Pros | Cons | Best For |
|------|------|------|----------|
| Fixed monthly fee | Predictable revenue, easy budgeting | Scope creep risk | Steady-state accounts |
| Per-square-foot | Scales with building size | Doesn't account for frequency changes | Office buildings |
| Per-hour (T&M) | Transparent, adjustable | Revenue unpredictable | Variable-scope work |
| Bundled services | Higher value per contract | Complex management | Full-service accounts |

### Scope Documentation
- **Clearly define scope in the contract**: List every task, area, and frequency. Use a scope matrix:

| Task | Daily | 3x/Week | Weekly | Monthly | Quarterly |
|------|-------|---------|--------|---------|-----------|
| Trash removal | ✓ | | | | |
| Vacuum carpet | | ✓ | | | |
| Mop hard floors | ✓ | | | | |
| High dusting | | | | | ✓ |
| Strip and wax | | | | | ✓ |

- Ambiguity in scope = future disputes. Be specific.
- Include change order procedures for scope additions.

### Client Communication
- Monthly or quarterly account review meetings.
- Respond to complaints within 2 hours, resolve within 24 hours.
- Proactive communication: "We noticed increased foot traffic. We recommend adding a midday restroom check." This builds trust and provides upsell opportunities.
