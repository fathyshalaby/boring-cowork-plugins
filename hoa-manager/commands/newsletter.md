# Command: /hoa:newsletter

Generate a community newsletter template.

## Output Format
```
[COMMUNITY NAME] NEWSLETTER
[Month Year]

FROM THE BOARD:
[Message from the board president — updates, gratitude, vision]

UPCOMING EVENTS:
• [Event, date, location, details]

MAINTENANCE UPDATES:
• [Current and upcoming maintenance projects]

REMINDERS:
• [Seasonal reminders — landscaping, parking, pool rules, etc.]

FINANCIAL SNAPSHOT:
Operating Fund Balance: $[X]
Reserve Fund Balance: $[X]
Delinquency Rate: [X]%

BOARD MEETING:
Next Meeting: [date, time, location]
All homeowners welcome!

CONTACT:
Management: [name, phone, email]
Emergency After-Hours: [phone]
Website: [URL]
```
