# Command: /hoa:arb-response

Generate an architectural review decision letter.

## Output Format
```
[Date]
[Owner Name and Address]

Re: Architectural Modification Request — [Address/Lot #]
    Application Date: [date]

Dear [Owner Name],

The Architectural Review Committee has reviewed your request
for the following modification:

PROPOSED MODIFICATION: [description]

DECISION: ☐ APPROVED  ☐ APPROVED WITH CONDITIONS  ☐ DENIED

[If conditions]: The following conditions must be met:
1. [condition]

[If denied]: Reason for denial:
[Specific governing document reference and standard not met]

NOTE: Work must be completed within [X] days of this approval.
A final inspection is required upon completion.

Questions: Contact [name] at [phone/email]

Sincerely,
Architectural Review Committee
```
