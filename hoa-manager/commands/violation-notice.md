# Command: /hoa:violation-notice

Generate a formal violation notice for a homeowner.

## Output Format
```
[Date]
[Owner Name and Address]

Re: Notice of Violation — [Address/Lot #]

Dear [Owner Name],

During a recent [inspection/complaint investigation], the following
violation of the community's governing documents was identified:

VIOLATION: [Description]
GOVERNING DOCUMENT REFERENCE: [CC&Rs Section X / Rule X]
DATE OBSERVED: [date]

REQUIRED ACTION:
[Specific corrective action needed]

COMPLIANCE DEADLINE: [date]

FAILURE TO COMPLY:
If the violation is not corrected by the deadline above, the
Association may impose fines of $[X] per [day/occurrence] and/or
take further enforcement action per the governing documents.

Your right to a hearing before the Board is available upon written
request within [X] days of this notice.

Sincerely,
[HOA Manager / Board President]
```
