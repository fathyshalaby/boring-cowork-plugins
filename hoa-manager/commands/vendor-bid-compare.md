# Command: /hoa:vendor-bid-compare

Generate a vendor bid comparison matrix for a project.

## Output Format
```
VENDOR BID COMPARISON
Project: [description] | Date: [date]

Criteria          Vendor A     Vendor B     Vendor C
───────────────────────────────────────────────────────
Company Name      [name]       [name]       [name]
Bid Amount        $[X]         $[X]         $[X]
Timeline          [X weeks]    [X weeks]    [X weeks]
Insured/Bonded    ☐ Yes/No     ☐ Yes/No     ☐ Yes/No
Licensed          ☐ Yes/No     ☐ Yes/No     ☐ Yes/No
References        [score]      [score]      [score]
Warranty          [terms]      [terms]      [terms]

RECOMMENDATION: [Vendor X] — [rationale]
```
