# Command: /trial:protocol-summary

Generate a protocol synopsis for site staff training.

## Output Format
```
PROTOCOL SYNOPSIS
Protocol: [Title]
Protocol Number: [X] | Version: [X] | Date: [date]
Sponsor: [name] | Phase: [I/II/III/IV]

OBJECTIVE: [Primary objective in 1-2 sentences]

DESIGN: [Randomized/open-label/double-blind, parallel/crossover, etc.]

POPULATION:
Target Enrollment: [X] subjects | Age: [range]
Key Inclusion: [top 3-5 criteria]
Key Exclusion: [top 3-5 criteria]

TREATMENT:
Arm 1: [treatment, dose, route, frequency, duration]
Arm 2: [treatment, dose, route, frequency, duration]

ENDPOINTS:
Primary: [endpoint]
Secondary: [endpoints]

VISIT SCHEDULE:
Visit    Day/Week    Key Assessments
──────────────────────────────────────────
Screen   -14 to -1   [assessments]
Baseline Day 0       [assessments]
Visit 1  Week 4      [assessments]
[etc.]

SAFETY MONITORING:
[SAE reporting timelines, DSMB requirements, stopping rules]
```
