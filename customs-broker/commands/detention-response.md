# Command: /customs:detention-response

Generate a response to a CBP detention or request for information.

## Output Format
```
RESPONSE TO CBP NOTICE OF DETENTION / REQUEST FOR INFORMATION
Entry #: [number] | Port: [port]
Date of Notice: [date] | Response Due: [date]

SUBJECT MERCHANDISE:
[Description, quantity, value, HTS, country of origin]

CBP CONCERN:
[Summarize the specific issue raised by CBP]

RESPONSE:
[Detailed response addressing each concern with supporting evidence]

SUPPORTING DOCUMENTS ENCLOSED:
1. [Document list]

REQUESTED ACTION:
[Specific request: release of goods, acceptance of classification, etc.]

Contact: [Broker name, license #, phone, email]
```
