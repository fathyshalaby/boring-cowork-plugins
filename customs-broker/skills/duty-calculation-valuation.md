# Duty Calculation & Valuation
Calculate duties accurately using proper valuation methods:

### Transaction Value (Primary Method)
- Duty = Entered Value × Duty Rate
- **Transaction value** = Price actually paid or payable + additions (assists, royalties, packing costs, proceeds of resale accruing to seller).
- Deductions: International freight, insurance, and other post-importation costs (if included in invoice price and separately identified).

### Alternative Valuation Methods (in order of preference)
1. Transaction value of identical merchandise
2. Transaction value of similar merchandise
3. Deductive value (selling price minus post-import costs)
4. Computed value (cost of production + profit + expenses)
5. Fallback method

### Special Duty Programs
| Program | Benefit | Requirement |
|---------|---------|-------------|
| GSP | Duty-free | Qualifying developing country, meets rules of origin |
| USMCA (formerly NAFTA) | Reduced/zero duty | Certificate of origin, rules of origin met |
| CAFTA-DR | Reduced/zero duty | Central American/DR origin |
| Section 301 (China) | Additional 7.5-25%+ | Chinese origin goods on Lists 1-4A |
| AD/CVD | Additional duties | Antidumping/countervailing duty orders |

### Reconciliation
- Use the Reconciliation program for entries where final value, classification, or USMCA eligibility isn't known at time of entry.
- Flag entries for reconciliation at time of filing. File the reconciliation entry within 21 months.
