# Command: /claims:status-update

Generate a professional claim status update for the insured.

## Output Format
```
CLAIM STATUS UPDATE
Date: [date]
Claim #: [number] | Date of Loss: [date]
Insured: [name]

CURRENT STATUS: [Investigation / Evaluation / Settlement / Closed]

PROGRESS UPDATE:
[Clear summary of what has been done since last communication]

NEXT STEPS:
1. [What will happen next and approximate timeline]
2. [Any action required from the insured]

INFORMATION NEEDED FROM YOU:
☐ [Any outstanding documentation or information required]

ESTIMATED TIMELINE:
[Expected date for next milestone or resolution]

QUESTIONS?
Contact: [Adjuster name]
Phone: [number] | Email: [email]
Claim is available online at: [portal URL]
```
