# Event Planning & Timeline Management
Plan events methodically from concept to execution:

### Event Planning Timeline
| Timeframe | Tasks |
|-----------|-------|
| 6-12 months out | Define objectives, set budget, select venue, book keynote speakers/entertainment |
| 4-6 months | Vendor RFPs (catering, AV, decor, photographer), finalize theme/branding, launch marketing |
| 2-3 months | Menu finalization, room layout/floor plan, registration opens, order printed materials |
| 1 month | Confirm all vendor details, create run-of-show (ROS), final headcount to caterer, brief staff |
| 1 week | Final walkthrough of venue, confirm deliveries, pack event supplies, print name badges |
| Day-of | Load-in at [time], sound check, vendor coordination, registration setup, run the show |
| Post-event | Vendor settlements, attendee survey, debrief meeting, final budget reconciliation |

### Run-of-Show (ROS) Template
The ROS is the minute-by-minute script for the event:
```
TIME        ACTIVITY                   RESPONSIBLE    NOTES
06:00 AM    Vendor load-in begins      Event Mgr      Loading dock access
08:00 AM    AV sound check             AV Company      Test mics, projector
09:00 AM    Registration opens         Staff (3)       Badges, swag bags
09:30 AM    Welcome remarks            Emcee           Cue lights dim
09:45 AM    Keynote speaker            Speaker name    60 min + 15 Q&A
[etc.]
```

### Risk Planning
- **Weather (outdoor events)**: Always have a rain plan. Tent rental on hold or indoor backup space.
- **No-shows**: Budget for 10-15% attrition for free events, 5% for paid events.
- **Vendor cancellation**: Have backup contacts for critical vendors (catering, AV) and require cancellation clauses in contracts.
