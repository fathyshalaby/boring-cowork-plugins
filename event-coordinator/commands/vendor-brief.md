# Command: /events:vendor-brief

Generate a comprehensive vendor brief/specification document for RFP distribution.

## Output Format
```
VENDOR BRIEF
Event: [name] | Date: [date]
Venue: [name, address]
Expected Attendance: [X]

SERVICE REQUIRED: [Catering / AV / Decor / Photography / etc.]

EVENT OVERVIEW:
[2-3 paragraph description of the event, theme, and audience]

SPECIFIC REQUIREMENTS:
• [Detailed list of what you need from this vendor category]

LOGISTICAL DETAILS:
Setup Time: [date/time]
Event Time: [start - end]
Teardown: [deadline]
Load-in Access: [details]
Power/Utilities: [what's available]

BUDGET RANGE: $[X] - $[X] (all-inclusive)

PROPOSAL MUST INCLUDE:
1. Detailed scope of services
2. Itemized pricing
3. References from similar events
4. Portfolio/samples
5. Insurance certificates

PROPOSAL DEADLINE: [date]
QUESTIONS TO: [name, email, phone]
```
