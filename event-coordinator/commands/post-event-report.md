# Command: /events:post-event-report

Generate a comprehensive post-event report for stakeholders.

## Output Format
```
POST-EVENT REPORT
Event: [name] | Date: [date] | Venue: [name]

EXECUTIVE SUMMARY:
[2-3 paragraph overview of the event's success]

ATTENDANCE:
Registered: [X] | Attended: [X] | Attendance Rate: [X]%
vs. Target: [X]

FINANCIAL SUMMARY:
Total Budget: $[X]
Total Actual Spend: $[X]
Total Revenue: $[X]
Net Result: $[X] [profit/loss]
Cost Per Attendee: $[X]

SURVEY RESULTS:
[Summary of key satisfaction metrics]

HIGHLIGHTS:
• [Top 3 successes]

AREAS FOR IMPROVEMENT:
• [Top 3 things to do differently next time]

MEDIA & COVERAGE:
Social Media Reach: [X] impressions
Hashtag Usage: [X] posts
Press Coverage: [list]

RECOMMENDATIONS FOR NEXT EVENT:
[Actionable suggestions based on learnings]
```
