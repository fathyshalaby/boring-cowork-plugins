# Building Code Awareness
Be aware of common building code requirements to identify significant deficiencies. Note: property inspectors report on conditions, they do NOT perform code compliance inspections.

### Safety-Critical Code Items to Flag
- **Smoke detectors**: Required in every bedroom, outside each sleeping area, and on every level. Interconnected preferred.
- **Carbon monoxide detectors**: Required on every level with a fuel-burning appliance or attached garage (varies by jurisdiction).
- **Stairway handrails**: Required on any stairway with 4+ risers. Must be graspable (1.25"–2" diameter).
- **Deck ledger attachment**: The ledger board must be lag-bolted or through-bolted to the rim joist. Nailed-only ledgers are a collapse hazard.
- **Tempered glass**: Required within 24" of a door, in bathrooms, and below 18" from floor. Check for the tempered glass etching.

### Reporting Approach
- **Do NOT cite specific code sections** unless you are a licensed code inspector.
- **DO report the condition** and note that it does not meet current safety standards.
- Example: "The deck ledger board appears to be attached with nails only, which does not meet current prescriptive standards for deck attachment. Recommend evaluation by a licensed contractor."

### Grandfather Clauses
- Homes are generally not required to meet current code unless renovated. However, always flag life-safety items regardless of the home's age.
