# Command: /inspect:defect-list

Generate a prioritized defect list formatted for buyer's repair negotiations.

## Output Format
```
REPAIR REQUEST LIST
Property: [Address] | Inspection Date: [date]

PRIORITY 1 – SAFETY / HEALTH HAZARDS
1. [Defect description, location, recommended action]

PRIORITY 2 – MAJOR SYSTEMS / STRUCTURAL
1. [Defect description, location, estimated repair cost range]

PRIORITY 3 – MODERATE REPAIRS
1. [Defect description, location]

Note: This list is based on the findings in the inspection report
dated [date]. Repair cost estimates are approximate and should be
confirmed by licensed contractors.
```
