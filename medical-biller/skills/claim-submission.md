# Claim Submission & Processing
Submit clean claims that get paid on the first attempt:

### Clean Claim Checklist
Before submitting any claim, verify:
- [ ] Patient demographics correct (name, DOB, gender, address match insurance records)
- [ ] Insurance ID number and group number correct
- [ ] Provider NPI (National Provider Identifier) correct
- [ ] Place of service (POS) code correct (11 = office, 21 = inpatient, 22 = outpatient, 23 = ER)
- [ ] Date of service correct
- [ ] Diagnosis codes support medical necessity for the procedures billed
- [ ] CPT codes correctly match the documentation
- [ ] Modifiers applied correctly
- [ ] Prior authorization number included (if required)
- [ ] Referring provider NPI (if required)
- [ ] Accident information (if applicable—auto accident, work comp, third party)

### Claim Forms
- **CMS-1500**: Used for professional (physician) claims. Submitted to commercial payers and Medicare Part B.
- **UB-04 (CMS-1450)**: Used for institutional (hospital/facility) claims. Submitted to Medicare Part A and payers for facility charges.

### Timely Filing
- Every payer has a timely filing deadline: Medicare (12 months), Medicaid (varies by state, typically 90 days–1 year), Commercial (varies—check contract, typically 90 days–1 year).
- **Track filing deadlines meticulously.** A missed deadline is money lost with no recourse.

### First-Pass Payment Rate
- Target: 95%+ clean claim rate (claims paid without rework).
- Track your denial rate by denial reason code. The top 3 reasons usually account for 60%+ of denials.
