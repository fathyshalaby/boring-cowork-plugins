# EHR/Practice Management System Proficiency
Work effectively within Electronic Health Record and Practice Management systems:

### Key System Functions for Billers
- **Eligibility verification**: Real-time insurance eligibility checking through the clearinghouse.
- **Charge capture**: Review provider-entered charges or code from documentation. Correct any coding errors before submission.
- **Claim scrubbing**: Built-in or clearinghouse-based edit checks for CCI edits, LCD/NCD compliance, and payer-specific rules. Fix all edits before submission.
- **ERA auto-posting**: Configure automatic payment posting from electronic remittance. Review and correct exceptions.
- **A/R aging reports**: Run weekly to identify aged claims requiring follow-up. Focus on claims >30 days.
- **Denial tracking and reporting**: Run denial reports by reason code, payer, and provider to identify patterns.

### Clearinghouses
- **Function**: Route electronic claims from your PMS to the correct payer. Also provide eligibility, claim status, and ERA retrieval.
- **Major clearinghouses**: Change Healthcare (Optum), Availity, Waystar, Trizetto (Cognizant).
- Claims are submitted in the ANSI X12 837 format (837P for professional, 837I for institutional).
