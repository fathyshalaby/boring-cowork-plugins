# Medical Coding Fundamentals (ICD-10, CPT, HCPCS)
Master the three primary medical code sets used in billing:

### ICD-10-CM (Diagnosis Codes)
- Used to report the medical reason for the encounter.
- Format: 3-7 alphanumeric characters (e.g., E11.65 = Type 2 diabetes with hyperglycemia).
- **Code to the highest level of specificity.** Using an unspecified code (e.g., J06.9 "Upper respiratory infection, unspecified") when documentation supports a specific diagnosis will trigger denials.
- **Sequencing matters**: The primary (first-listed) diagnosis is the main reason for the encounter. Secondary diagnoses justify medical necessity for additional services.
- Common mistake: Coding symptoms when the provider has documented a definitive diagnosis.

### CPT Codes (Procedure Codes)
- Developed by the AMA. Used to report medical, surgical, and diagnostic services.
- **E/M codes (99201-99215, 99281-99285)**: Evaluation and Management—the bread and butter of office visits. Level selection is based on Medical Decision Making (MDM) complexity OR total time.
- **Surgical codes**: Include a global period (0, 10, or 90 days). Services within the global period are typically included in the surgical fee.
- **Modifiers**: Essential for accurate billing. Common modifiers: -25 (significant, separately identifiable E/M on same day as procedure), -59 (distinct procedural service), -26 (professional component only), -TC (technical component only).

### HCPCS Level II
- Used primarily for durable medical equipment (DME), supplies, drugs, and services not covered by CPT.
- Format: Letter + 4 digits (e.g., J1885 = Ketorolac injection, E0601 = CPAP ventilator).
- Critical for Medicare and Medicaid billing.
