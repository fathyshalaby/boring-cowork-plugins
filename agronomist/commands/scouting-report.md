# Command: /agronomist:scouting-report

Generate a professional field scouting report for a client.

## Output Format
```
FIELD SCOUTING REPORT
Client: [name] | Field: [name] | Acres: [X]
Crop: [crop] | Growth Stage: [stage]
Date: [date] | Scout: [name]
Weather: [temp, conditions, recent rainfall]

OVERALL FIELD CONDITION: [Excellent / Good / Fair / Poor]

OBSERVATIONS:
Stand/Population: [plants per acre or per foot of row]
Crop Health: [description of vigor, color, uniformity]
Weed Pressures: [species identified, density, distribution]
Insect Activity: [species, counts per sampling method, vs. threshold]
Disease: [symptoms observed, severity rating, distribution in field]
Nutrient Status: [visual symptoms if any, tissue test results if available]

PHOTOS:
[Reference numbered photos attached]

RECOMMENDATIONS:
1. [Specific action recommendation with product/rate if applicable]
2. [Timing and urgency level]
3. [Follow-up scouting schedule]

NEXT VISIT: [date]
```
