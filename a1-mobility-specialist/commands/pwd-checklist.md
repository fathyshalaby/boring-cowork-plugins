# Command: /mobility:pwd-checklist

Generate compliance requirements for a specific host country posting.

## Output Format
```
POSTED WORKER DIRECTIVE (PWD) COMPLIANCE CHECKLIST
Host Country: [e.g., FRANCE]
Action Type: [e.g., Provision of IT Consulting Services for 3 months]

PRE-DEPLOYMENT (Must be completed BEFORE travel)
☐ 1. A1 Certificate Application submitted in Home Country.
☐ 2. Prior Declaration of Posting (SIPSI): Submit via the French SIPSI portal.
   - Required data: Start/end dates, client details, worker details, gross salary.
☐ 3. Appoint a Representative in France: (Must be reachable during the posting to present documents to labor inspectorate).
☐ 4. Salary Check: Ensure gross pay >= French "SMIC" (or applicable Syntec Collective Bargaining Agreement minimum) for the hours worked in France.

DOCUMENTS REQUIRED ON-SITE (Must be in FRENCH)
Worker must carry or local rep must immediately produce:
☐ A1 Certificate
☐ Copy of the SIPSI Declaration with QR code.
☐ Employment Contract (translated to French).
☐ Payslips covering the period of posting (translated).
☐ Timesheets showing actual hours worked in France.

ALLOWANCES NOTE
Per Diems meant to cover travel/board/lodging CANNOT be counted towards the French minimum wage calculation. Only allowances tied strictly to the work itself (e.g., hardship allowance) can be counted.
```\n
