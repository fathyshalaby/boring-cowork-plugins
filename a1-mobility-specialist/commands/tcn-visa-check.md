# Command: /mobility:tcn-visa-check

Screen a Third Country National posting for potential Vander Elst Visa requirements.

## Output Format
```
TCN POSTING IMMIGRATION SCREENER
Employee Nationality: [e.g., Indian]
Home State (Residence/Employer): [e.g., Poland]
Host State (Posting Destination): [e.g., Germany]
Duration: [e.g., 2 Months]

1. EU SOCIAL SECURITY (Reg 1231/2010)
- Does Reg 1231 apply between these states? [YES]
- Action: Apply for A1 from Polish ZUS.

2. IMMIGRATION / VANDER ELST ASSESSMENT
- General Principle: TCNs holding a Polish work/residence permit normally need a Visa to work in Germany.
- Vander Elst Exemption: Germany offers an exemption from the Vander Elst visa IF the posting is strictly temporary, based on a service contract between the Polish employer and German client, AND the duration is under 90 days in a 12-month period.

3. OUTCOME & REQUIREMENTS
- Duration Check: [60 days] is under the 90-day German limit.
- Visa Required? [NO, likely exempt under German rules].
- Mandatory Alternative Action: The employer MUST still notify the German Federal Employment Agency (Bundesagentur für Arbeit) Prior to the start of the provision of services.

*DISCLAIMER: Vander Elst implementation varies wildly by Member State (e.g., Austria requires a specific "Entsendebestätigung", Belgium strictly limits exemptions). Always verify current national immigration law.*
```\n
