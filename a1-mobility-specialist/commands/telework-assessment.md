# Command: /mobility:telework-assessment

Assess if a remote worker qualifies for the multilateral telework agreement.

## Output Format
```
FRAMEWORK AGREEMENT (CROSS-BORDER TELEWORK) ASSESSMENT

EMPLOYEE DATA
- Country of Residence: [e.g., France]
- Country of Employer: [e.g., Switzerland]
- Expected Working Pattern: [e.g., 40% remote in France, 60% on-site in Switzerland]

ELIGIBILITY CHECKLIST
1. Signatory Status:
   - Has Residence State signed the Framework Agreement? [YES - France]
   - Has Employer State signed the Framework Agreement? [YES - Switzerland]
2. Telework Definition:
   - Is the work in the residence state purely location-independent IT/remote work? [YES]
   - Any client visits/manual labor in residence state? [NO]
3. Threshold Check:
   - Is telework in residence state between 25% and 49.9%? [YES - 40%]
   - Is remaining work (min 50%) in the employer state? [YES - 60%]

CONCLUSION
☐ ELIGIBLE. The employee can opt-in to remain in the [Swiss] social security system despite exceeding the standard 25% rule in [France].

NEXT STEPS
- Submit A1 application to [Swiss SVA] explicitly referencing the Framework Agreement.
- Draft an Addendum to Employment Contract documenting the exact % split to satisfy future audits.
```\n
