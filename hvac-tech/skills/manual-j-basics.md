# Manual J Load Calculations
Understand and apply ACCA Manual J load calculations for proper system sizing:

### What is Manual J?
Manual J is the ANSI/ACCA-recognized standard for calculating heating and cooling loads for residential buildings. **Proper sizing is the foundation of every HVAC installation.**

### Why It Matters
- **Oversized systems**: Short-cycle, don't dehumidify properly, waste energy, and wear out faster.
- **Undersized systems**: Run constantly, can't maintain setpoint on design days, and generate complaints.
- **Rule of thumb ("1 ton per 400 sq ft") is NEVER acceptable.** It ignores insulation, windows, orientation, climate zone, and occupancy.

### Key Inputs
| Factor | What to Measure |
|--------|----------------|
| Building envelope | Wall insulation R-value, ceiling insulation R-value, floor type (slab, crawl, basement) |
| Windows | Type (single, double, low-E), area, orientation (N/S/E/W), shading |
| Infiltration | Blower door test result (ACH50) or visual assessment of air sealing quality |
| Climate | ACCA design temperatures for the location (99% heating, 1% cooling) |
| Internal loads | Occupants, appliances, lighting |
| Ductwork | Location (conditioned vs. unconditioned space), insulation level, leakage rate |

### Applying the Results
- Match equipment capacity to the **calculated load**, not the existing equipment size.
- Total capacity should be within 15% of the calculated load. Going 25% over is a costly oversizing mistake.
- In humid climates (Southeast US), slight undersizing is actually preferable for better dehumidification.
- Always document the Manual J calculation and provide it to the customer and local code inspector.
