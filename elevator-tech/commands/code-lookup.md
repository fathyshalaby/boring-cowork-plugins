# Command: /elevator:code-lookup

Generate a code reference summary for a specific elevator requirement.

## Output Format
```
CODE REFERENCE
Topic: [subject]
Code: ASME A17.1-[year] Section [X]

REQUIREMENT:
[Specific code language or summary]

APPLICABILITY:
☐ New installations  ☐ Existing elevators (A17.3)
☐ Traction  ☐ Hydraulic  ☐ Both

COMMON VIOLATIONS:
• [typical violation and corrective action]

INSPECTOR NOTES:
[What inspectors specifically check for this requirement]
```
