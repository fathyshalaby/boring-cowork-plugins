# Elevator Code & Standards
Master elevator codes and safety standards:

### Key Codes
| Code | Scope |
|------|-------|
| ASME A17.1 / CSA B44 | Safety Code for Elevators and Escalators (primary US/Canadian code) |
| ASME A17.2 | Inspections of Elevators (guide for inspectors) |
| ASME A17.3 | Safety Code for Existing Elevators |
| ASME A17.5 | Elevator and Escalator Electrical Equipment |
| ADA | Accessibility requirements (car size, controls, signals) |
| Local AHJ codes | Jurisdiction-specific amendments and requirements |

### Code Categories
- **Electric traction elevators**: Geared (low-rise, <250 fpm) and gearless (high-rise, 500-2000+ fpm).
- **Hydraulic elevators**: Holeless, holed, roped-hydro. Typical for 2-6 story buildings, <150 fpm.
- **Machine-room-less (MRL)**: Traction elevator with machine in the hoistway. Growing market share.
- **Escalators & moving walks**: ASME A17.1 Part 6.

### Inspection Categories
- **Category 1 (Annual)**: Routine inspection. Visual checks, operational tests, measurements.
- **Category 3 (Periodic)**: More thorough. Includes car top and pit inspections, safety device tests.
- **Category 5 (5-year)**: Full-load safety test, governor test, buffer test. Most comprehensive.
- **Acceptance inspection**: Before a new or modernized elevator enters service.
