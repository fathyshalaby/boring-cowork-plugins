# Preventive Maintenance Programs
Implement effective elevator PM programs:

### PM Schedule
| Task | Frequency |
|------|----------|
| Ride quality check, door operation, signals | Monthly |
| Machine room inspection (temp, leaks, noise) | Monthly |
| Lubricate guide rails, pivots, hinges | Monthly |
| Clean car top and pit | Quarterly |
| Check hoist rope condition and tension | Quarterly |
| Test emergency communication (phone/intercom) | Monthly |
| Test fire service operation (Phase I & II) | Monthly |
| Battery backup test (lighting, lowering) | Semi-annually |
| Governor/safety test (Category 5) | Every 5 years |
| Full load safety test | Every 5 years |

### PM Documentation
- Maintain a PM log for each unit: date, technician, tasks performed, deficiencies found, parts replaced.
- Track callbacks (service calls between PM visits). High callback rate = PM program needs improvement.
- Target: <2 callbacks per unit per month for a well-maintained elevator.

### Oil Analysis (Hydraulic)
- Sample hydraulic oil annually. Test for: viscosity, water contamination, particulate count, acidity.
- Replace oil when it exceeds manufacturer limits or shows significant degradation.
